Explain to non-tech friend

parentheses()

parentheses indicate a function

brackets[]

Brackets indicate arrays.

braces{}

Curly braces enclose code to be executed like a seperate paragraph in writing. Specific code statements such as If are followed by code written inside curly braces. Code inside curly braces are like a seperate section of code.

single quotes ''

single quotes are like double quotes indicating a string. Single quotes are used instead of double quotes to differentiate. Makes code easier to read at glance

double quotes ""

Double quotes are like single quotes indicating a string. Double quotes are used instead of single quotes to differentiate. Makes code easier to read at glance
