Recursion compares to iteration

Style of looping comes down to which is faster or which is easier to maintain.

Recursion: See my previous remark about "recursion". For more information you can also check out "recursion".

Iteration: After reading this word aloud, in between counting from 0 to 1000, you should be able to understand what this means.

A recursive function works through the process of calling itself until a condition is met. Iteration uses a looping control structure (while, do while, for) to repeat a section of code until a condition is met.
